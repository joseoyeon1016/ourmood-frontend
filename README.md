# 🎨 Virtual Fitting System

AI 기반 가상 피팅 시스템 (CatVTON)

## 기능
- 실시간 의상 착용 시뮬레이션
- AutoMasker 자동 마스킹
- 웹 인터페이스

## 기술 스택
- Backend: Flask, PyTorch, CatVTON
- Frontend: Next.js, React, Tailwind CSS
